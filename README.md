# career-kit

A giftable job-search copilot: [career-ops](https://github.com/santifer/career-ops)
plus a kit overlay that adds a guided intake interview, Simplify-assisted
applications in a dedicated Chrome profile, a durable answer bank, and a
code-enforced never-submit gate. Built to be dropped into the Claude app's
Code tab and driven entirely by conversation.
